2.	Stack
Your task is to implement the ADS IAbstractStack<Т> inside the Stack<Т> class provided.

You have to implement all the methods in order to solve the problem, 
however you are free to add more methods with any access modifier you want.

•	void Push(T item) 
o	Adds an element at the top of the stack and increases the size.

•	T Pop() 
o	Removes an element at the current top of the stack and returns it. 
o	If the stack is empty throw InvalidOperationException with appropriate message.

•	T Peek() 
o	Returns the element at the current top of the stack without removing it. 
o	If the stack is empty throw InvalidOperationException with appropriate message.

•	int Count 
o	Returns the number of elements inside the stack.
